Capstone Project - Vacation Planner App

Description

This is an Android application developed using Java in Android Studio. The app was built using Gradle and Room is used as an abstraction layer over SQLite for database functionality. JUnit was used to conduct a few tests.
Users can create an account and save vacations and associated excursions to a database. Vacations and excursions can be modified or deleted from the database, and each vacation is specific to an individual user by using an account ID. The app also features search functionality, sharing and notification features, and the ability to generate a report.

What I learned

The foundation of this application was created in my Mobile Applications class and then modified to fit the capstone requirements. This was the first Java project I built from start to finish. While previous classes had me modifying sections of code, this project allowed me to experience building an application from start to finish. I really enjoyed learning what went into a mobile application and had a lot of fun seeing the immediate results appear on my phone.
I got to experience how every aspect of an application worked together - from the database to what appears on the phone screen. I had to consider several different ways that the app would need validation functionality, such as preventing an end date to be before a start date. I also learned quite a bit about mobile development as well, such as RecyclerViews and notification features.
If I had to build this app a second time, there is still plenty that I would need to look up again to remember how to do it. However, I feel that it would be a much faster process and go much more smoothly.
